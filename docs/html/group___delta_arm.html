Detailed Description

Control module for the vehicle.

<feedback.h>

Here lies the heart and soul of the operation. feedback_init(void) pulls in the control constants from settings module and sets up the discrete controllers. From then on out, feedback_march(void) should be called by the IMU interrupt at feedback_hz until the program is shut down. feedback_march(void) will monitor the setpoint which is constantly being changed by setpoint_manager(void).
