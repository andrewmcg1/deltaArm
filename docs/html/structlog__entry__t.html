Detailed Description

Struct containing all possible values that could be writen to the log. For each log entry you wish to create, fill in an instance of this and pass to add_log_entry(void). You do not need to populate all parts of the struct. Currently feedback.c populates all values and log_manager.c only writes the values enabled in the settings file.
