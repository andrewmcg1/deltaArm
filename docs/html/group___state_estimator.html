Detailed Description

Navigation module for the vehicle.

<state_estimator.h>

This runs at the same rate as the feedback controller. state_estimator_march() is called immediately before feedback_march() in the IMU interrupt service routine.

Typedef Documentation

◆ state_estimate_t

typedef struct state_estimate_t state_estimate_t

This is the output from the state estimator. It contains raw sensor values and the outputs of filters. Everything is in NED coordinates defined as:

  • X pointing Forward
  • Y pointing Right
  • Z pointing Down

right hand rule applies for angular values such as tait bryan angles and gyro

  • Positive Roll to the right about X
  • Positive Pitch back about Y
  • Positive Yaw right about Z
