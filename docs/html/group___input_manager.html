Detailed Description

A DSM radio input manager.

<input_manager.h>

Contains Functions to start and stop the input manager thread which is the translation beween control inputs from DSM to the user_input struct which is read by the setpoint manager.

◆ dsm_kill_mode_t

enum dsm_kill_mode_t

Determines how the dsm radio indicates an arm/disarm kill switch.

Enumerator
DSM_KILL_DEDICATED_SWITCH 

A dedicated channel is used as a kill switch.

Carefully set the dsm_kill_ch and dsm_kill_pol channel and polarity settings.

DSM_KILL_NEGATIVE_THROTTLE 

Negative throttle is used as the kill switch.

This is the preferred method because it frees up a channel for other use. When using this mode, dsm_kill_ch and dsm_kill_pol are ignored.

Some radios, such as Spektrum DXe have an ARM/DISARM switch which forces the throttle channel down below normal range to disarm.
