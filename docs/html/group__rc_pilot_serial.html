Detailed Description

Serial communication module for the delta arm.

<rc_pilot_serial.h>

Date
29-11-2023
Author
Andrew McGrellis (andre.nosp@m.wmcg.nosp@m.@vt.e.nosp@m.du)

The delta arm functions on a seperate beaglebone than the quadcopter's flight controller so in order to use one xbee modem, the delta arm and quadcopter must communicate over serial. This module is responsible for reading packets from the serial port and parsing them into current and desired locations for the delta arm.
