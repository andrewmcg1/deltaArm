Detailed Description

Code for reading the NTP timer offset in order to beaglebones. Only used if "log_ntp" flag is 'true' in settings file and if NTP server is running on HP laptop and BeagleBoneBlue is running NTP client and both laptop and BeagleBoneBlue are connected to A2sys router.
